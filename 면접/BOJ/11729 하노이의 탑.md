분할정복법을 사용하는 하노이의 탑



**핵심**

원반 n개를 옮기기위해서는 n-1개의 원반을 옮겨야 한다.



우선 n개의 탑을 옮기기위한 횟수를 알아보자.

- 맨 아래 원반을 옮기기위해 우선 `n-1`개의 원반을 옮겨야한다.
  - `P(n-1)`번의 이동횟수
- 맨 아래 원반을 옮긴다.
  - `1`번의 이동횟수
- `P(n-1)`개의 원반을 맨 아래 원반으로 옮긴다.
  - `n-1`번의 이동횟수



원반을 옮기는 횟수는 `2 * P(n-1) + 1`의 횟수로 옮겨짐을 알 수 있다.

`P(1) = 1`

`P(2) = 2 * P(1) + 1 = 2 * 1 + 1 = 3`

`P(3) = 2 * P(2) + 1 = 2 * 3 + 1 = 7`

...

위처럼 계산됨을 알 수 있다.



**구현**

a에서 b로 원반을 옮겨야한다고 가정하자.

`left`는 a, b를 제외한 남는 기둥

- `n-1`개의 원반을 `left`로 옮긴다.
- `n`번째 원반을 `b`로 옮긴다.
- `n-1`개의 원반을 `b`로 옮긴다.



```python
n = int(input())

def hanoi(n, a, left, b):
    if n == 1:
        print(a, b)
        return
    
    hanoi(n - 1, a, b, left)
    hanoi(1, a, left, b)
    hanoi(n - 1, left, a, b)


print(2 ** n - 1)
hanoi(n, 1, 2, 3)
```
